Bauhaus Echo is now available on the [App Store](https://apps.apple.com/us/app/bauhaus-echo/id6759010657), [Google Play](https://play.google.com/store/apps/details?id=com.suchsoftware.bauhausecho), [itch.io](https://suchsoftware.itch.io/bauhaus-echo), and [the web](https://bauhaus.such.software). It is a visual memory game where you memorize tile arrangements, watch them shuffle, and reconstruct the original pattern.

## The Concept

The game shows you a grid of colorful geometric tiles inspired by Bauhaus design. You get a few seconds to memorize their positions. Then the tiles spin, shrink, and scatter to the bottom of the screen. Your job is to drag them back to where they belong.

## Four Game Modes

- **Classic**: the core experience. Memorize and reconstruct.
- **Speed**: tiles appear one at a time. Place each before the next drops.
- **Endless**: speed mode that loops forever. See how long you last.
- **Memory**: traditional card-matching pairs.

Each mode supports 2×2 through 5×5 grids and three difficulty levels. Global leaderboards let you compete against other players across all platforms.

## 500 Bauhaus Tiles

The tile collection features 500 hand-curated geometric designs drawn from the Bauhaus aesthetic: bold shapes, primary colors, clean lines. Each tile has beveled 3D edges and the game board is styled like a wood-and-felt craft table.

## Tech Stack

Built with [libGDX](https://libgdx.com/), a cross-platform Java game framework. One codebase runs on Android, iOS (via RoboVM), desktop (LWJGL3), and web (via GWT). The tile assets are packed into a TextureAtlas for efficient GPU rendering.

## Monetization

The mobile versions are free with AdMob banner ads. A one-time in-app purchase removes ads permanently. Desktop is a paid download ($2) with no ads. The web version is free.
